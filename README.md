## About The Project

A fullstack MERN application that allows users to keep track of their income and expenses for the month and year. Displays the data in a clean and modern graphical UI.

Here's why:
* I keep a manual transaction excel sheet and wanted to build and application that aggregates all the data together to output graphical data.
* Practice building fullstack applications using MERN
* Build fundamental understanding of working with REST API's
* Practice content layout and displaying data to users

### Built With
* [ReactJS](https://reactjs.org/)
* [Express](https://expressjs.com/)
* [MongoDB](https://www.mongodb.com/)
* [Node.js](https://nodejs.org/en/)
